# IMDb Sentiment Analysis using LSTM

## Overview

This project implements a sentiment analysis model for IMDb movie reviews using Long Short-Term Memory (LSTM) neural networks.

The model classifies reviews as positive or negative based on textual content.

## Technologies

* Python
* TensorFlow / Keras
* LSTM Networks
* Natural Language Processing (NLP)

## Files

* `app.py` — Application interface
* `imdb_lstm_Yashsarin.h5` — Trained LSTM model
* `imdb_word_index.json` — Tokenizer vocabulary
* `requirements.txt` — Project dependencies

## Run

```bash
pip install -r requirements.txt
python app.py
```
